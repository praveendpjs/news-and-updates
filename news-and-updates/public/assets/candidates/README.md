# Candidate Images Directory

Place candidate photos in the following directory structure:

```
public/assets/candidates/
├── placeholder.jpg (fallback image for missing photos)
└── [YEAR]/
    └── [SERVICE]/
        └── [candidate-name].jpg
```

## Example

For IFoS candidates from 2008:
```
public/assets/candidates/2008/ifos/
├── sasimohan.jpg
├── ganesh-karthik.jpg
├── nithya-kalyani.jpg
├── senbagapriya.jpg
├── kanchana.jpg
├── naveen-kumar.jpg
├── ramasamy.jpg
├── prabhakaran.jpg
├── david-camus.jpg
├── arthanari.jpg
├── sakthivel.jpg
└── jesin.jpg
```

## Image Requirements

- Format: JPG or PNG
- Recommended size: 100x120 pixels (portrait orientation)
- Aspect ratio: 3:4 (width:height)
- Background: Solid color (preferably white or light blue)
