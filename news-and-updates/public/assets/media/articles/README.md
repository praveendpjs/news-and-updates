## Supported Image Formats

- JPG/JPEG
- PNG
- WebP

## Image Requirements

- Recommended resolution: Minimum 800px width
- File size: Optimize images to reduce load time
- Aspect ratio: Maintain original newspaper clipping aspect ratio

## Adding New Images

1. Add the image file to this directory with the appropriate name
2. Update `media.json` in `src/app/Components/media-and-press/` with the new entry
3. Use the naming convention: `[topic]-[month]-[year].jpg`
